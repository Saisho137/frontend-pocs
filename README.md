realizados durante 2024. Cada carpeta es un experimento aislado con su propio stack.

|-----|-------|----------|
| `angular-reactive-forms` | Angular | Reactive Forms |
| `angular-interceptors` | Angular | HTTP Interceptors |
| `angular-guards` | Angular | Route Guards |
| `guards-react` | React Router | Equivalente de Guards en React |
| `react-hook-form` | React | react-hook-form |
| `react-contentful` | React + Contentful | CMS headless |
| `gatsby-contentful` | Gatsby + Contentful + GraphQL | Static site con CMS |
| `dark-mode` | HTML / CSS | Light–dark theme |
| `lazy-app` | Angular/React | Lazy loading patterns |
| `micro-frontends-udemy` | React | Curso Udemy de microfrontends |
| `react-cero-a-experto-udemy` | React + MERN | Curso Udemy React desde cero |
