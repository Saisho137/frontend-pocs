# Frontend POCs

Recopilación de pruebas de concepto (POCs) y ejercicios formativos en frontend.
Cada carpeta es un experimento aislado con su propio stack.

| POC | Stack | Concepto |
|-----|-------|----------|
| `angular-reactive-forms` | Angular | Reactive Forms |
| `angular-interceptors` | Angular | HTTP Interceptors |
| `angular-guards` | Angular | Route Guards |
| `angular17-template` | Angular 17 | Template con features de Angular 17 |
| `microfronts-angular-vite` | Angular + Vite | Arquitectura de microfrontends |
| `transactional-app` | Angular + RxJS | Clean Architecture con patrones de diseño |
| `state-handlers` | Redux / Context / NGRX | Comparativa de gestores de estado |
| `guards-react` | React Router | Equivalente de Guards en React |
| `react-hook-form` | React | react-hook-form |
| `react-contentful` | React + Contentful | CMS headless |
| `gatsby-contentful` | Gatsby + Contentful + GraphQL | Static site con CMS |
| `dark-mode` | HTML / CSS | Light–dark theme |
| `lazy-app` | Angular/React | Lazy loading patterns |
| `micro-frontends-udemy` | React | Curso Udemy de microfrontends |
| `react-cero-a-experto-udemy` | React + MERN | Curso Udemy React desde cero |
| `advanced-css-styles` | CSS | Animaciones, transiciones light-dark, transformaciones |
| `poc-astro` | Astro | Capacidades de Astro para sitios content-first |
